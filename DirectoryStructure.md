Directory Structure
========

GitBucket persists all data into HOME/gitbucket. This directory has following structure:

 * /HOME/gitbucket
   * /repositoties
     * /USER_NAME
       * / REPO_NAME.git
   * /tmp
     * /USER_NAME
       * /branches
         * /REPO_NAME
           * /BRANCH_NAME
       * /init-REPO_NAME
